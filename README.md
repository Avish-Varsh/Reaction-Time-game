# Reaction-Time-game
<br>
<br>
This is a Reaction time game that utilises a led chaser to imporve the reaction time of the user in a fun manner. Along with helping begginers learn key arduino and coding concepts. 
<br>
<br>
This game utilises 8 LEDS (2 LEDS must be of the same color and another pair of LEDS can't be of the same color) , 8 220 OHM resistors, 2 Push buttons, 1 Potentiometer, 1 Piezo buzzer,  1 Arduino UNO and Some Jumper Wires. 
<br>
<br>
Once the the arduino is powered on the LEDs connected to PINS 0 and 1 will light up to show that there is power. After both of these LEDs turn of a random LED will turn on which the user will have to catch in the LED chaser. Once the user remembers the LED they must press the start button causing the chaser to start. The user the use the potentiometer to control the speed of the chaser. They user will have to use the catch button to catch the LED. Depending on the outcome either a losing or winning sequence will play.
